# ModularPress Core

A modern modular CMS admin panel for PHP 8 projects, focused on clean architecture, predictability and performance.

![ModularPress Admin](./assets/modularpress-preview.png)

---

## Overview

ModularPress is not a theme and not a plugin marketplace.  
It is a focused CMS admin panel designed for teams that value structure, clarity and long term maintainability.

The system is built around modular architecture, clear separation of responsibilities and predictable behavior.

---

## Key Features

- Modular and predictable architecture
- Clean PHP 8 codebase
- Clear separation between content, users and configuration
- Performance oriented design
- Built for real world production projects

---

## Use Cases

- Corporate websites
- Service company websites
- Custom admin panels
- Long term maintained projects
