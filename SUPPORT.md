# Support for ModularPress

ModularPress is a commercial product. Support is provided through official channels, not through GitHub issues or pull requests.

---

## Where to get help

For any of the following:

- Product questions  
- Bug reports  
- Installation or upgrade help  
- Module configuration questions  
- Licensing and billing topics

Use one of these:

- ModularPress contact form, https://modularpress.eu/contact  
- The dedicated support email or portal provided in your commercial agreement

Mention your company name, project name and relevant environment details so the issue can be processed faster.

---

## What not to use this repository for

Please **do not**:

- Open GitHub issues for support  
- Send pull requests with fixes or features  
- Use this repository as a general discussion forum

Such items may be closed without response, in order to keep the repository clean and focused on documentation.

---

## Documentation

Public documentation and product overview live on:

- https://modularpress.eu

This repository may contain technical notes and examples, but the authoritative source of product information is the main site and your commercial documentation.
